                  # 3.4 解决方案（set names gbk;）

                  通过 set names gbk 设置下 MySQL 的编码为 GBK，设置后结果如下：

                  此方式只是在当前 mysql 环境下起作用，退出 mysql 再进入则恢复成 utf8
